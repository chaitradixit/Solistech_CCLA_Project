Field Descriptions

All fields marked with a * must have a value, or an error will be thrown when attempting to add the audit entry.

lApp* The Stelllent component/application where the audit entry is coming from.
lAction* The action being audited. Use short, unique codes for each action type.
lRef* Generally refers to the dDocName of the item being audited.
lTitle* Generally refers to the title of the item being audited.
lUser* Name of the user performing the action
lMessage A description of the action. 500 character limit.
lParam1 - lParam6 A set of optional data parameters that provide extra information for the audit action. 100 character limits.
